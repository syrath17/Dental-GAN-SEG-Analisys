# Dental-GAN-SEG-Analisys
This project aims at developing a GAN model to generate synthetic ortopanoramic dental X-ray images, and training and evaluating a YOLO-based segmentation model on real and synthetic datasets. The main focus is to assess how well a segmentation model trained on real data generalizes to synthetic images and to identify its potential limitations.


This repository includes:
- Training and testing codes;
- A link to the datasets used;
- Project presentation
